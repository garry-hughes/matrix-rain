# Matrix Rain

[![skills.sh](https://skills.sh/b/garry/matrix-rain)](https://skills.sh/garry/matrix-rain)

Matrix-style green code rain animation for the terminal — installable as an AI agent skill.

## Installation

```bash
npx skills add garry/matrix-rain
```

## What It Does

Renders the iconic Matrix "falling code" animation directly in PowerShell. Columns of random characters cascade down the terminal with bright white heads and dark green trails.

## Usage

```powershell
pwsh ./skills/matrix-rain/scripts/matrix-rain.ps1
```

Press `Ctrl+C` to stop.

## Requirements

- PowerShell 5.1+ (Windows) or PowerShell 7+ (macOS/Linux)
